Animating probability distributions

Beta distribution

x = collect(0.01:0.01:0.99);
α = collect(0.1:0.2:10);
β = collect(0.1:0.5:10);
animate_beta(x, α, β)

Normal distribution

x = collect(-4:0.01:4);
μ = collect(-4:0.5:4);
σ = collect(1:0.1:8);
animate_normal(x, μ, σ)

Gamma distribution

x = collect(0.1:0.01:4);
α = collect(0.1:0.1:5);
θ = collect(0.5:0.1:2);
animate_gamma(x, α, θ)

Cauchy distribution

x = collect(-4:0.01:4);
μ = collect(-4:1:4);
σ = collect(1:0.1:8);
animate_cauchy(x, μ, σ)

Weibull distribution

x=collect(0:0.01:6);
α=collect(0.1:0.1:5);
θ=collect(1:0.25:10);
animate_weibull(x, α, θ)

Inverse-Gamma distribution

x = collect(0.1:0.01:4);
α = collect(0.1:0.1:5);
θ = collect(0.5:0.1:2);
animate_invgamma(x, α, θ)

Frechet distribution

animate_frechet(x, α, θ)

Rayleigh distribution

x=collect(0:0.01:5);

Exponential distribution

Pareto distribution

x=collect(0:0.01:6);

animate_pareto(x, α, θ)

Tri-weight distribution

animate_triweight(x, μ, σ)

Student's t distribution

x=collect(-4:0.01:4);
ν=collect(0.5:0.1:5); # letter "nu", not "v"
animate_tstudent(x, ν)
